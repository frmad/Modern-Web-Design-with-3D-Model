Exploring Modern Web Design with 3D Models

This site is a personal project built to experiment with cutting-edge web design practices, specifically focusing on using HTML, CSS, and JavaScript to integrate 3D models into web pages. The butterfly model you see here is powered by the Three.js library.

About This Project

ABOUT

This website is an experiment to better understand modern web development techniques, with a focus on integrating interactive 3D models into user-friendly designs. Using technologies like Three.js, this project demonstrates how JavaScript libraries can enhance web visuals and interactions.

Technologies Used

HTML, CSS & JavaScript

The foundation of this project is built with standard web technologies: HTML for structure, CSS for styling, and JavaScript for interactivity and animation.

Three.js

Three.js is used to load and animate the 3D butterfly model on this website, allowing for smooth rendering and responsive control.

Web Development

This project emphasizes modern design practices, creating visually appealing and user-friendly interfaces that enhance the overall user experience.
